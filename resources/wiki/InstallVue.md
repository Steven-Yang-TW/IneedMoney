## 先安裝Vue Js
 ```php
 composer require laravel/ui
 ```

 再者把vue 部屬在 laravel 上
 ```php
  php artisan ui vue
 ```

然後編譯npm
```javascript
npm install 
```
```javascript
npm run dev
```
